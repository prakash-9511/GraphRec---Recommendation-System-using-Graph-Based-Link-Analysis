Algorithm Pipeline
01
Load Amazon Reviews Dataset
Reads reviews.csv, products.csv, users.csv into Spark DataFrames with defined schemas. 200 users, 100 products, ~2000 reviews.
02
Build Weighted User→Product Graph
Edge weight = rating (1–5) + verified bonus (0.5) + helpful votes ratio (0–0.5). Captures quality of interaction.
03
Project to Product↔Product Co-interaction Graph
Self-join on user_id: two products are linked if the same user rated both. Edge weight = product of both weights.
04
Run Iterative PageRank (10 iterations, d=0.85)
rank(v) = (1−d)/N + d × Σ[rank(u) × w(u,v) / Σw(u,·)]. Products co-purchased with influential products rank higher.
05
Generate Recommendations
For a user, find unrated products. Score = co_interaction_weight × PageRank. Return top-N by final score.
